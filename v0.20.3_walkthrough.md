# Walkthrough - v0.20.3 Drilldown / Evidence Inspection

This milestone introduces the drilldown layer for consensus review, enabling researchers to perform detailed inspections of individual comparisons directly within the Streamlit workspace.

## Key Accomplishments

### 1. Comparison Detail Pane (Drilldown)
Added an "Inspection & Drilldown" pane below the main review table:
- **Smart Selection**: A selectbox that automatically tracks the current filtered results.
- **Decision Metrics**: At-a-glance summary of status, labels, support margins, and conflict flags.
- **Reference Tables**: Detailed lists of supporting and conflicting reference comparisons, including their integrated scores and ranks.
- **Artifact Tracking**: Direct display of relative paths to related files (Decisions, Evidence, Summary, Handoff).

### 2. Troubleshooting with Raw Data
- **Raw JSON Inspection**: Integrated expanders to view the underlying `st.json` for both Decision and Evidence Profile payloads. This allows reviewers to verify the source data for any controversial comparison.

### 3. Refactored Load Flow
- **One-Time Bundle Intake**: Optimized the loading logic in `comparator_review_sections.py` to keep the `ConsensusBundleImportContext` in memory. This eliminates redundant file reads and enables deep inspection without additional overhead.
