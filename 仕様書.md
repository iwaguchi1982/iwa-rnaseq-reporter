iwa-rnaseq-reporter 読込仕様 v0.1 draft
1. 位置づけ

iwa-rnaseq-reporter は、iwa-rnaseq-counter v0.1.7 が出力した reporter-ready output contract を読み込み、以下を行うアプリとする。

dataset 単位の読込
sample / QC / expression matrix の整合確認
reporter 用内部データモデルへの正規化
後段の可視化・統計処理・PDF/HTML レポート生成への受け渡し

reporter は counter の再計算エンジンではない。
基本責務は 「標準成果物を受け取り、解釈可能な dataset に再構成すること」 とする。

2. reporter の入口ファイル
2.1 正式な入口

reporter の入口は dataset_manifest.json とする。

理由:

counter 側で reporter 接続用入口として明示済み
reporter が最初に「何を読むべきか」を固定できる
後でファイル命名が少し増えても manifest で吸収できる
2.2 基本ルール

reporter はまず dataset_manifest.json を読む。
manifest に記載されたパス・ファイル一覧をもとに、必要成果物を順に解決する。

つまり reporter は原則として

いきなり gene_tpm.csv を直読みしにいかない
results/ を総当たりで推測しない
「manifest にあるか」を第一の真実とする

とする。

3. reporter が読む成果物
3.1 必須ファイル

最低限、以下を必須とする。

dataset_manifest.json
run_summary.json
sample_metadata.csv
sample_qc_summary.csv
gene_tpm.csv
gene_numreads.csv
必須にする理由

reporter の初期責務が

dataset の成立確認
sample 情報の保持
QC の要約表示
gene-level 可視化・解析

だからです。

特に reporter の PCA, correlation, DEG, heatmap の多くは、初期段階では gene matrix が中心 になるので、
まずは gene レベルを必須にしておくのが堅いです。

3.2 準必須ファイル

以下は あれば読む だが、なくても dataset 読込自体は成立とする。

transcript_tpm.csv
transcript_numreads.csv
位置づけ
transcript browser
isoform 的な補助表示
後で transcript search をやる時の基盤

に使えるので重要ですが、reporter v0.1.x の最初の成立条件には入れなくてよいです。

3.3 任意ファイル
sample_sheet.csv
run_config.json
logs/run.log
用途
provenance / 再現性確認
詳細 troubleshooting
付録レポート

これらは reporter の主計算には直接不要です。

4. ディレクトリ前提

counter v0.1.7 の固定 contract を前提にする。

results/<run_name>/
├── results/
│   ├── transcript_tpm.csv
│   ├── transcript_numreads.csv
│   ├── gene_tpm.csv
│   ├── gene_numreads.csv
│   ├── sample_metadata.csv
│   ├── sample_qc_summary.csv
│   ├── run_summary.json
│   └── dataset_manifest.json
├── sample_sheet.csv
├── run_config.json
└── logs/
    └── run.log

reporter は manifest の場所を基準に相対解決できる 実装にしておくのがよいです。
つまり、manifest 内の path が相対でも絶対でも読めるようにする。

5. reporter の読込モード
5.1 推奨モード
A. manifest 指定モード

ユーザーが dataset_manifest.json を指定する

これは正式モード。

B. run directory 指定モード

ユーザーが results/<run_name>/ を指定する

この場合 reporter 側で以下を推測する。

<run_dir>/results/dataset_manifest.json を探す
あればそれを入口にする
なければ読込不可
C. results directory 指定モード

ユーザーが results/<run_name>/results/ を指定する

この場合も

dataset_manifest.json
を探して入口にする。
5.2 非推奨モード

単独 CSV だけ読ませるモードは、少なくとも v0.1.x では主モードにしない方がいいです。

理由:

sample metadata と expression のズレを検知しにくい
provenance が消える
reporter の入力 contract を弱くする

必要なら将来

“matrix only quick viewer”
として別モード化した方がよいです。
6. reporter 内部データモデル

reporter は読み込んだファイルを、そのまま使うのではなく 内部標準モデル に正規化した方がいいです。

6.1 ReporterDataset

6.2 なぜ内部モデル化するか

理由は3つです。

1. UI と処理を分離できる

Streamlit 画面は ReporterDataset を受け取るだけで済む。

2. validation 結果を持てる

「読めたが注意あり」を dataset に保持できる。

3. 後方互換を吸収しやすい

将来 manifest schema が v0.1.7 -> v0.2.0 に変わっても、
loader 側で吸収できる。

7. sample ID を中心にした整合ルール

ここはかなり重要です。
reporter は sample_id 整合性チェッカー でもあるべきです。

7.1 基本ルール

以下の集合を比較する。

sample_metadata.csv の sample_id
sample_qc_summary.csv の sample_id
gene_tpm.csv の列名 sample 群
gene_numreads.csv の列名 sample 群
run_summary.json の success / failed / aggregated sample 群
原則

集約対象 sample = reporter の主 dataset sample 群 とする。

つまり reporter で本計算に使うのは基本的に

aggregated_sample_ids
または
successful_sample_ids

です。

7.2 推奨ルール
A. gene matrix の列が最優先の事実

gene_tpm.csv と gene_numreads.csv の列 sample を主集合とする。

理由:

reporter の統計処理は結局この行列に依存するため
B. metadata / qc はそこへ左寄せで揃える
gene matrix にある sample は metadata に存在すべき
gene matrix にある sample は qc summary に存在すべき
C. 余分 sample は warning

metadata や qc にだけ存在して、matrix に存在しない sample は warning 扱い。

これは部分成功 run で起こり得るので、即エラーにはしない。

7.3 エラー条件

以下は読込失敗にしてよいです。

sample_metadata.csv に sample_id 列がない
sample_qc_summary.csv に sample_id 列がない
gene_tpm.csv と gene_numreads.csv の列 sample 集合が一致しない
gene matrix に sample 列が1つもない
manifest / summary が壊れて主要ファイル解決不能
8. gene / transcript matrix のフォーマット規約

reporter 側で明文化した方が後々楽です。

8.1 gene matrix
gene_tpm.csv
行: gene_id
列: sample_id
値: numeric float
gene_numreads.csv
行: gene_id
列: sample_id
値: numeric float or int-like
8.2 transcript matrix
transcript_tpm.csv
行: transcript_id
列: sample_id
transcript_numreads.csv
行: transcript_id
列: sample_id
8.3 保存形式の前提

reporter 側では以下を想定して読むと安全です。

1列目が feature ID
1列目の列名は gene_id または transcript_id を推奨
sample 列は2列目以降
wide format

pandas 読込時はだいたいこういう方針で十分です。

df = pd.read_csv(path)
feature_col = df.columns[0]
df = df.set_index(feature_col)

9.3 追加列

それ以外の列も保持してよい。

15. 読込アルゴリズムの擬似コード
def load_reporter_dataset(manifest_path):
    manifest = load_manifest(manifest_path)
    messages = validate_manifest_schema(manifest)

    resolved = resolve_manifest_paths(manifest, manifest_path)
    messages += validate_required_files(resolved)

    if has_fatal(messages):
        raise ReporterLoadError(messages)

    run_summary = read_json(resolved["run_summary"])
    sample_metadata = pd.read_csv(resolved["sample_metadata"])
    sample_qc_summary = pd.read_csv(resolved["sample_qc_summary"])
    gene_tpm = read_csv_indexed(resolved["gene_tpm"])
    gene_numreads = read_csv_indexed(resolved["gene_numreads"])

    transcript_tpm = None
    transcript_numreads = None
    if "transcript_tpm" in resolved and resolved["transcript_tpm"].exists():
        transcript_tpm = read_csv_indexed(resolved["transcript_tpm"])
    else:
        messages.append(warn("missing_transcript_tpm", "transcript_tpm.csv is not available"))

    if "transcript_numreads" in resolved and resolved["transcript_numreads"].exists():
        transcript_numreads = read_csv_indexed(resolved["transcript_numreads"])
    else:
        messages.append(warn("missing_transcript_numreads", "transcript_numreads.csv is not available"))

    sample_metadata = normalize_sample_metadata(sample_metadata)
    sample_qc_summary = normalize_sample_qc_summary(sample_qc_summary)
    gene_tpm = normalize_expression_matrix(gene_tpm)
    gene_numreads = normalize_expression_matrix(gene_numreads)

    messages += validate_sample_metadata(sample_metadata)
    messages += validate_sample_qc_summary(sample_qc_summary)
    messages += validate_expression_matrix(gene_tpm, "gene_id")
    messages += validate_expression_matrix(gene_numreads, "gene_id")
    messages += validate_sample_id_consistency(
        sample_metadata, sample_qc_summary, gene_tpm, gene_numreads, run_summary
    )

    if has_fatal(messages):
        raise ReporterLoadError(messages)

    sample_ids = list(gene_tpm.columns)

    return ReporterDataset(
        dataset_id=manifest["dataset_id"],
        run_name=manifest["run_name"],
        manifest_path=Path(manifest_path),
        base_dir=Path(manifest_path).parent.parent,
        results_dir=Path(manifest_path).parent,
        manifest=manifest,
        run_summary=run_summary,
        sample_metadata=sample_metadata,
        sample_qc_summary=sample_qc_summary,
        gene_tpm=gene_tpm,
        gene_numreads=gene_numreads,
        transcript_tpm=transcript_tpm,
        transcript_numreads=transcript_numreads,
        sample_ids=sample_ids,
        successful_sample_ids=run_summary.get("successful_samples", []),
        failed_sample_ids=run_summary.get("failed_samples", []),
        aggregated_sample_ids=run_summary.get("aggregated_samples", sample_ids),
        messages=messages,
    )

def resolve_manifest_paths(manifest_path: str | Path, manifest_norm: dict) -> dict[str, Path]:
    manifest_path = Path(manifest_path).resolve()
    results_dir = manifest_path.parent
    run_dir = results_dir.parent

    resolved = {}
    for key, rel in manifest_norm["paths"].items():
        rel_path = Path(rel)
        if rel_path.is_absolute():
            resolved[key] = rel_path
        else:
            resolved[key] = run_dir / rel_path

    resolved["_manifest_path"] = manifest_path
    resolved["_results_dir"] = results_dir
    resolved["_run_dir"] = run_dir
    return resolved

def msg(level: str, code: str, message: str) -> ValidationMessage:

def validate_manifest_raw_legacy(manifest_raw: dict) -> list[ValidationMessage]:
    messages = []

    for key in ["app_name", "app_version", "run_name", "files"]:
        if key not in manifest_raw:
            messages.append(msg("fatal", "manifest_missing_key", f"dataset_manifest.json missing key: {key}"))

    files = manifest_raw.get("files", {})
    for key in ["sample_metadata", "sample_qc_summary", "gene_tpm", "gene_numreads", "run_summary"]:
        if key not in files:
            messages.append(msg("fatal", "manifest_missing_file_key", f"dataset_manifest.json files missing key: {key}"))

    return messages


def validate_required_files(resolved_paths: dict) -> list[ValidationMessage]:
    messages = []
    required = ["sample_metadata", "sample_qc_summary", "gene_tpm", "gene_numreads", "run_summary"]

    for key in required:
        path = resolved_paths.get(key)
        if path is None or not path.exists():
            messages.append(msg("fatal", "missing_required_file", f"Required file not found: {key}"))

    optional = ["transcript_tpm", "transcript_numreads", "sample_sheet", "run_config", "run_log"]
    for key in optional:
        path = resolved_paths.get(key)
        if path is None or not path.exists():
            messages.append(msg("warning", "missing_optional_file", f"Optional file not found: {key}"))

    return messages

    if len(df.columns) == 0:

def validate_matrix_pair(gene_tpm, gene_numreads) -> list[ValidationMessage]:
    messages = []
    if list(gene_tpm.columns) != list(gene_numreads.columns):
        messages.append(
            msg("fatal", "matrix_sample_mismatch", "gene_tpm.csv and gene_numreads.csv sample columns do not match")
        )
    return messages


def validate_sample_id_consistency(sample_metadata, sample_qc_summary, gene_tpm, run_summary: dict) -> list[ValidationMessage]:
    messages = []

    metadata_ids = set(sample_metadata["sample_id"].astype(str))
    qc_ids = set(sample_qc_summary["sample_id"].astype(str))
    matrix_ids = set(map(str, gene_tpm.columns))

    summary_all = set(map(str, run_summary.get("sample_ids_all", [])))
    summary_success = set(map(str, run_summary.get("sample_ids_success", [])))
    summary_failed = set(map(str, run_summary.get("sample_ids_failed", [])))
    summary_aggregated = set(map(str, run_summary.get("sample_ids_aggregated", [])))

    missing_in_metadata = matrix_ids - metadata_ids
    if missing_in_metadata:
        messages.append(msg("fatal", "matrix_ids_missing_in_metadata", f"Samples missing in metadata: {sorted(missing_in_metadata)}"))

    missing_in_qc = matrix_ids - qc_ids
    if missing_in_qc:
        messages.append(msg("fatal", "matrix_ids_missing_in_qc", f"Samples missing in QC summary: {sorted(missing_in_qc)}"))

    extra_metadata = metadata_ids - matrix_ids
    if extra_metadata:
        messages.append(msg("warning", "extra_metadata_samples", f"Metadata-only samples: {sorted(extra_metadata)}"))

    extra_qc = qc_ids - matrix_ids
    if extra_qc:
        messages.append(msg("warning", "extra_qc_samples", f"QC-only samples: {sorted(extra_qc)}"))

    if summary_aggregated and matrix_ids != summary_aggregated:
        messages.append(
            msg("warning", "aggregated_ids_mismatch", f"Matrix samples differ from run_summary sample_ids_aggregated: {sorted(matrix_ids)} vs {sorted(summary_aggregated)}")
        )

    if summary_success and not matrix_ids.issubset(summary_success):
        messages.append(msg("warning", "matrix_not_subset_success", "Matrix sample ids are not a subset of run_summary sample_ids_success"))

    if summary_failed & matrix_ids:
        messages.append(msg("warning", "failed_sample_in_matrix", "Failed samples are present in expression matrix"))

    if summary_all and not matrix_ids.issubset(summary_all):
        messages.append(msg("warning", "matrix_not_subset_all", "Matrix sample ids are not a subset of run_summary sample_ids_all"))

    return messages

import pandas as pd


def normalize_sample_metadata(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()

    if "sample_id" in df.columns:
        df["sample_id"] = df["sample_id"].astype(str)

    if "display_name" not in df.columns:
        df["display_name"] = df["sample_id"]
    else:
        df["display_name"] = df["display_name"].fillna("").astype(str)
        df.loc[df["display_name"].str.strip() == "", "display_name"] = df["sample_id"]

    if "condition" not in df.columns and "group" in df.columns:
        df["condition"] = df["group"]
    elif "condition" in df.columns and "group" in df.columns:
        cond_blank = df["condition"].fillna("").astype(str).str.strip() == ""
        df.loc[cond_blank, "condition"] = df.loc[cond_blank, "group"]

    if "exclude" in df.columns:
        df["exclude"] = (
            df["exclude"]
            .fillna(False)
            .astype(str)
            .str.strip()
            .str.lower()
            .map({"true": True, "false": False, "1": True, "0": False, "yes": True, "no": False, "": False})
            .fillna(False)
        )
    else:
        df["exclude"] = False

    if "replicate" in df.columns:
        df["replicate"] = df["replicate"].fillna("").astype(str)

    return df


def normalize_sample_qc_summary(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()
    if "sample_id" in df.columns:
        df["sample_id"] = df["sample_id"].astype(str)
    return df


def normalize_expression_matrix(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()
    df.columns = [str(c) for c in df.columns]
    for col in df.columns:
        df[col] = pd.to_numeric(df[col], errors="coerce")
    return df

class ReporterLoadError(Exception):
    def __init__(self, messages):
        self.messages = messages
        super().__init__("\n".join([f"[{m.level}] {m.code}: {m.message}" for m in messages]))


def has_fatal(messages) -> bool:
    return any(m.level == "fatal" for m in messages)


def load_reporter_dataset(manifest_path: str | Path) -> ReporterDataset:
    manifest_path = Path(manifest_path).resolve()

    messages = []

    manifest_raw = load_manifest(manifest_path)
    if "files" in manifest_raw:
        messages.extend(validate_manifest_raw_legacy(manifest_raw))

    manifest_norm = normalize_manifest(manifest_raw)
    resolved = resolve_manifest_paths(manifest_path, manifest_norm)

    messages.extend(validate_required_files(resolved))
    if has_fatal(messages):
        raise ReporterLoadError(messages)

    run_summary = read_json(resolved["run_summary"])

    sample_metadata = normalize_sample_metadata(read_csv(resolved["sample_metadata"]))
    sample_qc_summary = normalize_sample_qc_summary(read_csv(resolved["sample_qc_summary"]))
    gene_tpm = normalize_expression_matrix(read_csv_indexed(resolved["gene_tpm"]))
    gene_numreads = normalize_expression_matrix(read_csv_indexed(resolved["gene_numreads"]))

    transcript_tpm = None
    transcript_numreads = None

    if resolved.get("transcript_tpm") and resolved["transcript_tpm"].exists():
        transcript_tpm = normalize_expression_matrix(read_csv_indexed(resolved["transcript_tpm"]))

    if resolved.get("transcript_numreads") and resolved["transcript_numreads"].exists():
        transcript_numreads = normalize_expression_matrix(read_csv_indexed(resolved["transcript_numreads"]))

    messages.extend(validate_sample_metadata(sample_metadata))
    messages.extend(validate_sample_qc_summary(sample_qc_summary))
    messages.extend(validate_expression_matrix(gene_tpm, "gene_tpm"))
    messages.extend(validate_expression_matrix(gene_numreads, "gene_numreads"))
    messages.extend(validate_matrix_pair(gene_tpm, gene_numreads))
    messages.extend(validate_sample_id_consistency(sample_metadata, sample_qc_summary, gene_tpm, run_summary))

    if has_fatal(messages):
        raise ReporterLoadError(messages)

    dataset_id = manifest_norm["dataset_id"]
    run_name = manifest_norm["run_name"]
    analysis_name = manifest_norm["analysis_name"]

    sample_ids_all = list(map(str, run_summary.get("sample_ids_all", manifest_norm.get("sample_ids_all", []))))
    sample_ids_success = list(map(str, run_summary.get("sample_ids_success", manifest_norm.get("sample_ids_success", []))))
    sample_ids_failed = list(map(str, run_summary.get("sample_ids_failed", manifest_norm.get("sample_ids_failed", []))))
    sample_ids_aggregated = list(map(str, run_summary.get("sample_ids_aggregated", list(gene_tpm.columns))))

    return ReporterDataset(
        dataset_id=dataset_id,
        run_name=run_name,
        analysis_name=analysis_name,
        app_name=manifest_norm["app_name"],
        app_version=manifest_norm["app_version"],
        manifest_path=manifest_path,
        run_dir=resolved["_run_dir"],
        results_dir=resolved["_results_dir"],
        manifest_raw=manifest_raw,
        manifest_norm=manifest_norm,
        run_summary=run_summary,
        sample_metadata=sample_metadata,
        sample_qc_summary=sample_qc_summary,
        gene_tpm=gene_tpm,
        gene_numreads=gene_numreads,
        transcript_tpm=transcript_tpm,
        transcript_numreads=transcript_numreads,
        sample_ids_all=sample_ids_all,
        sample_ids_success=sample_ids_success,
        sample_ids_failed=sample_ids_failed,
        sample_ids_aggregated=sample_ids_aggregated,
        messages=messages,
    )
